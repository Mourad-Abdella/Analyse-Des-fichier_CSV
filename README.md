# Projet 3 - Analyse de fichier CSV

## Fonctionnalités
- Lecture d'un fichier CSV
- Calcul de la moyenne, minimum et maximum pour chaque colonne
- Affichage des résultats dans la console
- Génération d'un rapport CSV (resume.csv)

## Instructions
1. Placer le fichier `data.csv` dans le dossier racine du projet
2. Compiler et exécuter `Main.java`
3. Les résultats seront affichés dans la console et sauvegardés dans `resume.csv`
